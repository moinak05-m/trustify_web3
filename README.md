# Trustify Web3

Trustify Web3 is a decentralized application (dApp) that uses blockchain and smart contracts to securely store and verify data in a tamper-proof and transparent way.

---

## 🚀 What Problem It Solves
Traditional systems rely on centralized authorities for verification, which can be manipulated or forged. This project uses blockchain to ensure data cannot be altered once stored.

---

## ⚙️ How It Works
- Users connect their wallet (MetaMask)
- Data is submitted and stored via smart contracts
- Once stored, the data becomes immutable
- Anyone can verify the data directly from the blockchain

---

## 🛠 Tech Stack
- Solidity (Smart Contracts)
- Hardhat (Development & Deployment)
- React.js (Frontend)
- Ethers.js (Blockchain interaction)
- Ethereum (Blockchain network)
